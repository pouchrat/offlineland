You can use the following formatting options in Readable and Writable texts, as well as some of them in your Friends list status:

This is a Headline
==================

This is _emphasis_ (typically italics)...

this is **strong emphasis** (typically bold)...

this is ~~strikethrough~~...

this is ^^smaller or less important^^...

(You can escape the underscore, asterisk and tilde to not have this formatting be preceding with a backslash, like "\_".)

Color

Colorize text by wrapping with -colorname- or -colorname back- like this:

This is -green-some colorful-green- text...

This is -green back-some colorful background-green back- on text...

Rotation

Rotate a part of your text by using the following syntax, degrees ranging from 5-355 in steps of 5 (this is supported in Readables and Writables, but not the Friends list):

This is -rotate10-some rotated-rotate10- text...

This is -rotate180-some rotated-rotate180- text...

This is -rotate350-some rotated-rotate350- text...

Alignment

You can align your text to the center, or right, by using (in Readables and Writables, but not the Friends list):

    

-center-This is some text-center-



-right-This is some text-right-

Blink and Glow

You can make text blink and glow by using (in Readables and Writables, but not the Friends list):

    
-blink-This is some text-blink-

-blinksoft-This is some text-blinksoft-

-glow-This is some text-glow-

Mixed-case

If you want to show upper and lower-case letters, use:

THIS IS SOME TEXT. -case-This is some text.-case-
